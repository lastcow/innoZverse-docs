# Cybersecurity

Cybersecurity protects systems, networks, and data from attacks. This section covers both the defensive and offensive sides — from understanding how attacks work to how to stop them.

## Learning Path

### 🛡️ Fundamentals
Build your foundation: networking, cryptography, and attack vectors.

### ⚔️ Ethical Hacking
Learn to think like an attacker. Penetration testing methodology, OSINT, and exploitation.

### 🔍 Threat Intelligence
Analyze threats, respond to incidents, and implement security monitoring.
