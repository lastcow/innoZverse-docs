# Cybersecurity

Cybersecurity is one of the fastest-growing fields in tech. This section covers both defensive and offensive security concepts — from network fundamentals to ethical hacking.

## Learning Path

1. [Network Fundamentals](network-fundamentals.md)

## Tools You'll Use

- **Kali Linux** — Industry-standard security distribution
- **Wireshark** — Network packet analyzer
- **Nmap** — Network scanner
- **Metasploit** — Penetration testing framework
- **Burp Suite** — Web security testing
