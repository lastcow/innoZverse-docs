# Installation Guide

## Prerequisites

- Node.js 22 or newer
- Any operating system (macOS, Linux, Windows)
- ~500MB free disk space

## Install OpenClaw

### macOS / Linux

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

### Windows (PowerShell)

```powershell
iwr -useb https://openclaw.ai/install.ps1 | iex
```

## Verify Installation

```bash
node --version  # Should be 22 or higher
openclaw --version
```
