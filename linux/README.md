## ⚡ Quick Start

{% hint style="info" %}
**New to Linux?** Start at [Lab 1: Introduction to the Terminal](foundations/labs/lab-01-intro-to-terminal.md) — no prior experience needed. Each lab takes 20–30 minutes and includes verified command output.
{% endhint %}

{% hint style="warning" %}
**Prerequisites:** A terminal environment — Ubuntu 22.04 recommended. All labs are tested on Ubuntu 22.04 LTS. Use WSL2 on Windows or any Linux VM.
{% endhint %}
