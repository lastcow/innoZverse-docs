# Cloud & DevOps

Modern software runs in the cloud. DevOps bridges development and operations to deliver software faster and more reliably.

## Learning Path

### ☁️ Cloud Fundamentals
Understand cloud concepts, AWS core services, and Azure basics.

### 🐳 Containers & DevOps
Docker, Kubernetes, CI/CD pipelines, and GitHub Actions.

### 🏗️ Infrastructure as Code
Terraform and Ansible for automated, repeatable infrastructure.
