# HTML/CSS Foundations

**15 labs** from a blank HTML file to responsive layouts with Flexbox and Grid.

| # | Lab | Key Concepts |
|---|-----|-------------|
| 1 | [HTML Document Structure](labs/lab-01-html-document-structure.md) | DOCTYPE, head, body, semantic HTML |
| 2 | [Text Elements](labs/lab-02-text-elements.md) | headings, paragraphs, spans, strong, em |
| 3 | [Links & Images](labs/lab-03-links-and-images.md) | anchor, href, img, alt, paths |
| 4 | [Lists](labs/lab-04-lists.md) | ul, ol, dl, nested lists |
| 5 | [Tables](labs/lab-05-tables.md) | table, tr, th, td, colspan, rowspan |
| 6 | [Forms & Input Types](labs/lab-06-forms.md) | form, input, select, textarea, validation |

**No runtime needed** — pure HTML/CSS, open in any browser.
