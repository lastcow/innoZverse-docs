# JavaScript Foundations

Build a rock-solid understanding of JavaScript fundamentals. These 15 labs cover everything from your first `console.log` to async callbacks.

## 🎯 Track Goals

By the end of this track you will:
- Write and run JavaScript in Node.js confidently
- Understand variables, types, and type coercion
- Work with strings, arrays, and objects fluently
- Control program flow with conditionals and loops
- Write functions in all modern JS styles
- Understand scope, closures, and hoisting
- Handle errors gracefully
- Use ES modules
- Master functional array methods
- Apply destructuring and spread
- Create classes and use inheritance
- Understand callback-based async patterns
