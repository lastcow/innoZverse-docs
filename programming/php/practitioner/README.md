# PHP Practitioner

Go beyond the basics with advanced OOP patterns, PHP 8.x features, PDO repositories, REST APIs, functional pipelines, generators, and a full capstone MVC framework — all without external dependencies.

{% hint style="info" %}
**Prerequisites:** Complete PHP Foundations before starting Practitioner labs.
{% endhint %}

## Quick Start

{% tabs %}
{% tab title="🐳 Docker (Recommended)" %}
```bash
docker pull zchencow/innozverse-php:latest
docker run --rm zchencow/innozverse-php:latest php --version
```
{% endtab %}
{% tab title="🐧 Linux / macOS" %}
```bash
php --version   # requires PHP 8.1+
```
{% endtab %}
{% endtabs %}
