# Python Foundations

**15 labs** covering everything from your first `print()` to building a professional project structure. No experience required.

## Labs

| # | Lab | Key Concepts |
|---|-----|-------------|
| 1 | [Hello World & Basics](labs/lab-01-hello-world.md) | print, variables, arithmetic, f-strings |
| 2 | [Variables & Data Types](labs/lab-02-variables-types.md) | int, float, str, bool, type conversion |
| 3 | [Strings & String Methods](labs/lab-03-strings.md) | slicing, strip, split, join, format |
| 4 | [Lists, Tuples & Sets](labs/lab-04-lists-tuples-sets.md) | sequences, comprehensions, set operations |
| 5 | [Dictionaries](labs/lab-05-dictionaries.md) | key-value, Counter, defaultdict, JSON |
| 6 | [Control Flow](labs/lab-06-control-flow.md) | if/elif/else, for, while, break, continue |
| 7 | [Functions & Scope](labs/lab-07-functions.md) | def, args, kwargs, lambda, recursion |
| 8 | [File I/O](labs/lab-08-file-io.md) | open, csv, json, pathlib, context managers |
| 9 | [Error Handling](labs/lab-09-error-handling.md) | try/except/finally, custom exceptions |
| 10 | [Modules & Packages](labs/lab-10-modules-packages.md) | import, stdlib, itertools, functools |
| 11 | [Comprehensions & Generators](labs/lab-11-list-comprehensions.md) | list/dict/set comp, yield, lazy iteration |
| 12 | [OOP Basics](labs/lab-12-oop-basics.md) | class, self, inheritance, dataclass |
| 13 | [Debugging & Testing](labs/lab-13-debugging.md) | logging, assertions, unittest, pytest |
| 14 | [Regular Expressions](labs/lab-14-regex.md) | re module, patterns, validation, parsing |
| 15 | [Virtual Envs & Packaging](labs/lab-15-virtual-envs-packaging.md) | venv, pip, pyproject.toml, project structure |

All labs **Docker-verified** on Python 3.12.
