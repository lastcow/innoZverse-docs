# TypeScript Practitioner Labs

**Level:** Practitioner | **Labs:** 15 | **Prerequisites:** TypeScript basics, Node.js

This track builds solid TypeScript skills from the type system foundations through real-world application patterns. Complete all 15 labs to master practitioner-level TypeScript.

---

## Lab Index

| # | Lab | Topic | Key Skills |
|---|-----|-------|------------|
| 01 | [Type System Deep Dive](labs/lab-01-type-system-deep-dive.md) | Core type system | Union types, intersection types, type narrowing, `typeof`/`instanceof` |
| 02 | [Generics](labs/lab-02-generics.md) | Generic programming | Generic functions, classes, constraints, default types |
| 03 | [Advanced Interfaces & Types](labs/lab-03-advanced-interfaces-types.md) | Structural typing | Interface merging, `extends`, `implements`, optional/readonly |
| 04 | [Decorators](labs/lab-04-decorators.md) | Decorator pattern | Class, method, property, parameter decorators |
| 05 | [Async TypeScript](labs/lab-05-async-typescript.md) | Async patterns | `async/await`, typed Promises, error handling in async |
| 06 | [Strict Mode & Null Safety](labs/lab-06-strict-mode-null-safety.md) | Strictness | `strictNullChecks`, non-null assertion, optional chaining |
| 07 | [Enums & Namespaces](labs/lab-07-enums-namespaces.md) | Value types | `const enum`, string enums, namespace organization |
| 08 | [Advanced Classes](labs/lab-08-classes-advanced.md) | OOP patterns | Abstract classes, access modifiers, static members, mixins |
| 09 | [Modules & Paths](labs/lab-09-modules-paths.md) | Module system | ES modules, path aliases, barrel exports, `import type` |
| 10 | [Type Guards & Assertions](labs/lab-10-type-guards-assertions.md) | Type narrowing | Custom type guards, assertion functions, discriminated unions |
| 11 | [Zod Runtime Validation](labs/lab-11-zod-runtime-validation.md) | Schema validation | `z.object`, `z.infer`, `safeParse`, error formatting |
| 12 | [Express + TypeScript](labs/lab-12-express-typescript.md) | REST APIs | Typed routes, request/response types, middleware types |
| 13 | [Testing with Vitest](labs/lab-13-testing-vitest.md) | Testing | `describe/it/expect`, typed mocks, coverage |
| 14 | [tsconfig Advanced](labs/lab-14-tsconfig-advanced.md) | Compiler config | `strict`, `paths`, `lib`, `target`, compiler options |
| 15 | [Capstone: Typed API](labs/lab-15-capstone-typed-api.md) | Integration project | Full typed REST API with validation, tests, strict mode |
